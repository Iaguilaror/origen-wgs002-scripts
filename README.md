# origen-wgs002-scripts
Scripts and Data to reproduce figures of the oriGen-1427 WGS paper

## Description  
To promote reproducibility in comp. bio. works, we uploaded data slices and scripts to create the plots in main and supplementary figures of our manuscript:  TO-DO after publication.  

## How to use  

Go to the corresponding subdirectory, and there you can find R or python scripts, and source data. Run the scripts to get pdf, svg, or png images.  

It is important to note that some figures require sensitive data. In those cases, we cannot share the data publicly in this repository.

Some figures require big files, that exced the limit of github.
